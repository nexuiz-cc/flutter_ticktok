### フロントエンド

- **フレームワーク**: Flutter
- **状態管理**: Riverpod または Bloc
- **ルーティング**: go_router
- **ネットワーク**: dio
- **ビデオ再生**: video_player, chewie

### バックエンド

- **BaaSプラットフォーム**: Google Firebase
- **主要サービス**:
  - Firebase Authentication (認証)
  - Cloud Firestore (NoSQLデータベース)
  - Firebase Storage (ファイルストレージ)
  - Cloud Functions (サーバーレス関数)
  - Cloud Messaging (プッシュ通知)
## 2. アーキテクチャ設計

```
プレゼンテーション層 (UIコンポーネント)
├── ビジネスロジック層 (サービス/プロバイダー)
```

### コアコンポーネント

1. **ホームフィード**: 縦スクロール動画再生
2. **動画アップロード**: 撮影・編集・投稿機能
3. **ユーザープロフィール**: 個人ページ・作品管理
4. **ソーシャル機能**: コメント・いいね・フォロー
5. **検索・発見**: コンテンツ探索機能
